竹箒日記 : 2005/03

2005/3/7 : 大丈夫大丈夫。（きのこ）
イリヤの出番はちゃんとあるよー。

あと制作も良い感じで進行中だよー。

バイオ４は終わったけど遊んでばかりじゃないよー。

つーか休みがないよー。

ミクシィやりたいけどなんか色々面倒そうなんでやっぱりやめたよー。


あー、テニプリの映画観に行きてぇ。
2005/3/25 : まだ寒いけど温々の日（きのこ）
　土日ぐらいはゲームです。

　18禁ゲーかコンシューマーか悩んだあげく、寒かったんで温かそうな「ゆのはな」に走りました。
　おおう、ここのゲームは毎度いい作り込みなんですが、今回は一段と丁寧です。足回りしっかりしてるなあ。街の雰囲気いいなあ。つーか最近はどこもレベル高いなあ。

　そんないいゲームをコタツでまったりプレイできる幸せ。あー、温泉行きたいですなー。大江戸温泉物語じゃ我慢できねー。


　ところで軍艦……じゃなくて戦闘機とか戦車の話なのですが、コナミのメカ娘がこれまたいいデキなのですよ奥さん！
　武内が年甲斐もなく大人買いしてきたのですが（予約までしてやがった！）、横で見てたら我慢できなくなって閉店前のオモチャ屋に吶喊、自分も大人買いしてきちまったぜ。
　どいつもこいつもスッゲーいい出来です。
　顔の作りもちゃんと島田フミカネ絵っぽいよー、可愛いよー、ちゃんとローレグだよー！（←ここ重要）


　そうして美少女ゲーをやりながらメカ娘を組み立てつつコタツで眠る金曜の夜。
　オタクイズビューテフル……！



　日曜日は仕事に出てくださいネ☆
